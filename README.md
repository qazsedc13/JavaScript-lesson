# Урок 11: Функции

## Описание

Урок по изучению функций в JavaScript. Функции позволяют структурировать код и делают его многократно используемым.

## Технологический стек

- **Разметка:** HTML5
- **Логика:** JavaScript

## Структура проекта

- `index.html` — HTML-страница
- `js/JavaScript.js` — JavaScript-файл с кодом урока

## Установка и запуск

1. Откройте `index.html` в веб-браузере
