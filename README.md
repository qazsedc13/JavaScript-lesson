# Урок PHP-24: Функции даты

## Описание

Урок по изучению PHP. Функции для работы с датой и временем.

## Технологический стек

- **Backend:** PHP

## Структура проекта

- `mySite/www/` — директория с файлами проекта

## Установка и запуск

1. Требуется локальный сервер (Apache/PHP)
2. Разместите файлы в директории веб-сервера
3. Откройте в браузере
