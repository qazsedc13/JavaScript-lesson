# Урок MySQL-40: Выборка записей из БД

## Описание

Урок по работе с MySQL. Выборка записей из базы данных.

## Технологический стек

- **Backend:** PHP
- **База данных:** MySQL

## Структура проекта

- `mySite/www/` — директория с файлами проекта

## Установка и запуск

1. Требуется локальный сервер (Apache/PHP/MySQL)
2. Настройте подключение к базе данных
3. Разместите файлы в директории веб-сервера
4. Откройте в браузере
