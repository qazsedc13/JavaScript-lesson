# Урок PHP-29: Отправка почты на email

## Описание

Урок по изучению PHP. Отправка электронной почты.

## Технологический стек

- **Backend:** PHP

## Структура проекта

- `mySite/www/` — директория с файлами проекта

## Установка и запуск

1. Требуется локальный сервер (Apache/PHP)
2. Настройте почтовый сервер
3. Разместите файлы в директории веб-сервера
4. Откройте в браузере
