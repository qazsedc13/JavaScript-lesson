# Урок Ajax-6: Формат JSON

## Описание

Урок по изучению AJAX. Работа с форматом JSON для обмена данными.

## Технологический стек

- **Backend:** PHP
- **Frontend:** HTML, JavaScript
- **Технология:** AJAX, JSON

## Структура проекта

- `Ajax/mySite/www/` — директория с файлами проекта

## Установка и запуск

1. Требуется локальный сервер (Apache/PHP)
2. Разместите файлы в директории веб-сервера
3. Откройте в браузере
