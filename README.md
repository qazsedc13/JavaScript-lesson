# Урок PHP-27: phpinfo() и $_SERVER

## Описание

Урок по изучению PHP. Функция phpinfo() и суперглобальный массив $_SERVER.

## Технологический стек

- **Backend:** PHP

## Структура проекта

- `mySite/www/` — директория с файлами проекта

## Установка и запуск

1. Требуется локальный сервер (Apache/PHP)
2. Разместите файлы в директории веб-сервера
3. Откройте в браузере
