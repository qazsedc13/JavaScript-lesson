# Урок Ajax-4: Полный контроль

## Описание

Урок по изучению AJAX. Полный контроль над AJAX-запросами.

## Технологический стек

- **Backend:** PHP
- **Frontend:** HTML, JavaScript
- **Технология:** AJAX

## Структура проекта

- `Ajax/mySite/www/` — директория с файлами проекта

## Установка и запуск

1. Требуется локальный сервер (Apache/PHP)
2. Разместите файлы в директории веб-сервера
3. Откройте в браузере
