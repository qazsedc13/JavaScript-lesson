# Урок PHP-18: Подключение файлов

## Описание

Урок по изучению PHP. Подключение внешних файлов (include, require).

## Технологический стек

- **Backend:** PHP

## Структура проекта

- `mySite/www/` — директория с файлами проекта

## Установка и запуск

1. Требуется локальный сервер (Apache/PHP)
2. Разместите файлы в директории веб-сервера
3. Откройте в браузере
