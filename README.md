# Урок Ajax-5: Проверка логина

## Описание

Урок по изучению AJAX. Проверка логина на сервере без перезагрузки страницы.

## Технологический стек

- **Backend:** PHP
- **Frontend:** HTML, JavaScript
- **Технология:** AJAX

## Структура проекта

- `Ajax/mySite/www/` — директория с файлами проекта

## Установка и запуск

1. Требуется локальный сервер (Apache/PHP)
2. Разместите файлы в директории веб-сервера
3. Откройте в браузере
