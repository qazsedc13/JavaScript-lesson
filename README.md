# JavaScript Lesson Test

Минимальный проект для проверки работоспособности JavaScript в браузере.

## Описание

Данный проект содержит единственный HTML-файл, предназначенный для базовой проверки среды выполнения JavaScript. Он служит в качестве отправной точки для изучения клиентских скриптов и понимания работы тега `<noscript>`.

## Технологический стек

- **Разметка:** HTML5
- **Логика:** JavaScript (клиентский)

## Установка и запуск

1. Склонируйте репозиторий:
   ```bash
   git clone git@github.com:qazsedc13/JavaScript-lesson.git
   ```
2. Перейдите в папку `JavaScript-lesson/`.
3. Откройте файл `index.html` в любом современном веб-браузере.

## Примеры использования

- **Проверка JS:** Если на странице отображается текст "Is JavaScript working?", значит браузер поддерживает и выполняет скрипты.
- **Обработка отсутствия JS:** Если JavaScript отключен в настройках браузера, пользователь увидит сообщение "NO", выведенное тегом `<noscript>`.

## Структура проекта

- `index.html` — основной файл проекта с базовой структурой документа.
- `README.md` — описание проекта.

## Зависимости и требования

- Любой современный веб-браузер.
