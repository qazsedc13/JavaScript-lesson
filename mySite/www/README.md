# Оператор эквивалентности

Изучены операторы сравнения в PHP: `==` (равенство) и `===` (идентичность).

## Что изучили

- Оператор `==` проверяет равенство значений с приведением типов
- Оператор `===` проверяет и значение, и тип данных (строгая проверка)
- Примеры поведения: `0 == ""` возвращает `true`, но `0 === ""` возвращает `false`
